# Cycle Together

Мини-приложение `webxdc` для Arcane Chat / Delta Chat: один общий календарь цикла на чат.

## Что уже умеет

- хранить предыдущие циклы
- прогнозировать следующую менструацию и овуляцию
- вести ежедневные заметки о симптомах и самочувствии
- показывать календарь, историю и простые графики
- синхронизировать изменения между участниками через `webxdc`
- показывать иконку приложения в чатах, которые поддерживают `webxdc`-иконки
- экспортировать и импортировать резервную копию именно этого календаря

## Как смотреть результат во время разработки

1. Открой PowerShell в папке проекта.
2. Выполни:

```powershell
.\preview.bat
```

3. Браузер откроет `http://127.0.0.1:4173/index.html`
4. Для имитации второго участника открой вторую вкладку:

```text
http://127.0.0.1:4173/index.html?name=Партнер
```

В локальном режиме синхронизация идёт через браузерное хранилище, чтобы можно было проверять совместное редактирование ещё до упаковки в `.xdc`.
Сворачиваемый блок с кнопками для второй вкладки показывается только в локальном браузерном режиме и в самом чате не отображается.

## Как собрать `.xdc`

```powershell
.\build.bat
```

Готовый файл появится здесь:

```text
dist\cycle-together.xdc
```

## Как использовать в Arcane Chat / Delta Chat

1. Открой нужный диалог.
2. Прикрепи файл `dist\cycle-together.xdc` как обычный файл.
3. Отправь его в чат.
4. Открой присланный файл внутри чата.
5. После этого оба участника смогут вносить изменения в один и тот же календарь внутри этой копии приложения.

Важно: каждый отправленный `.xdc` в чате живёт как отдельный экземпляр приложения со своим собственным календарём. То есть один файл в одном диалоге = один общий календарь для этого диалога.

## Резервная копия

- В приложении есть блок `Экспорт и импорт`.
- `Экспортировать файл` создаёт JSON-файл только с данными этого календаря.
- `Отправить backup в чат` подготавливает этот же backup как файл для отправки в другой чат или в избранное, если клиент поддерживает `sendToChat()`.
- `Импортировать файл` восстанавливает backup в текущий календарь и заменяет его данные для всех участников этого `webxdc`.
